# 河洛战阵 / STZB Battle Prototype

一个静态 HTML/CSS/JS 的《率土之滨》风格战斗原型，聚焦三人编队、战法配置、抽卡拆解、官方资料导入和一回合一回合推进的 PVE 战斗。

## 快速开始

直接用浏览器打开 `index.html` 即可运行；项目没有构建步骤。

页面脚本加载顺序：

1. `src/battle-rules.js`：战斗规则常量、状态定义、伤害公式。
2. `official-data.js`：由脚本生成的官方武将/战法资料。
3. `src/seed-data.js`：本地种子武将、种子战法、官方别名映射。
4. `src/battle-engine.js`：回合推进、选目标、状态、伤害、治疗。
5. `src/app.js`：DOM、存档、编队、抽卡、弹窗、渲染。

## 当前功能

- 三武将编队：大营、中军、前锋，每名武将 10000 兵，最多 8 回合。
- 战斗按回合推进，不一次性跳到完整结果。
- 武将十连抽；3 星和 4 星自动转为狗粮，并解锁对应可拆战法。
- 武将册、战场单位、抽卡结果使用本地头像。
- 武将详情和战法详情弹窗可点击查看。
- 可配战法遵循队伍内唯一占用规则。
- 已接入准备回合、主动战法准备、洞察免疫、怯战、犹豫、混乱、暴走、援护、分兵、兵种克制、同阵营/同兵种加成、伤兵池。
