# 0008. SLG 主视图固定为天下地图

## 背景

项目已经从单场 PVE 战斗原型转为单机 SLG。旧的双方阵列战斗预览面板适合验证战斗引擎，但会抢占主屏焦点，让当前玩法看起来仍像单场战斗模拟器。

## 决策

- 页面默认主视图固定为“天下”大地图。
- 旧的战斗预览面板不再作为可见主界面展示。
- 出征战斗继续复用现有三人编队战斗引擎，结果进入战报弹窗和回放。
- 战报入口放入天下资源栏动作区，避免依赖旧顶栏按钮。

## 原因

- SLG 的核心操作是查看地块、发展主城、出征占领和结束回合，大地图应占据第一视觉层级。
- 战斗详情是出征后的回放和复盘信息，不应常驻在地图主视图里。
- 保留战报和回放能继续复用已有战斗表现，同时不让旧 PVE 面板干扰 SLG 主循环。

## 影响

- 浏览器烟测需要验证大地图是主视图，并确认旧战斗预览面板隐藏。
- 如果后续需要编队、武将册或战法库常驻入口，应作为地图外的管理层或抽屉设计，而不是恢复旧战斗预览面板。
